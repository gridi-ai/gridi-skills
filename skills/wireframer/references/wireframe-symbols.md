# Wireframe Symbol Guide

## Basic Containers

```
┌─────────────────────────────────────┐
│                                     │
│       Page/Section Container        │
│                                     │
└─────────────────────────────────────┘
```

## Input Fields

### Text Input
```
┌─────────────────────────────────────┐
│  Placeholder text                   │
└─────────────────────────────────────┘
```

### Labeled Input
```
Label
┌─────────────────────────────────────┐
│                                     │
└─────────────────────────────────────┘
```

### Text Area
```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│                                     │
└─────────────────────────────────────┘
```

## Buttons

### Primary Button
```
╔═══════════════════╗
║   Button Text     ║
╚═══════════════════╝
```

### Secondary Button
```
┌───────────────────┐
│   Button Text     │
└───────────────────┘
```

### Text Button
```
[ Button Text ]
```

## Selection Elements

### Checkbox
```
[ ] Unselected item
[x] Selected item
```

### Radio Button
```
( ) Unselected item
(o) Selected item
```

### Dropdown
```
┌─────────────────────────────────▼─┐
│  Selected value                    │
└───────────────────────────────────┘
```

## Navigation

### Header
```
┌─────────────────────────────────────┐
│ [Logo]          Menu1 Menu2   [👤] │
└─────────────────────────────────────┘
```

### Tabs
```
┌────────┬────────┬────────┐
│  Tab1  │  Tab2  │  Tab3  │
└────────┴────────┴────────┘
```

### Sidebar
```
┌────────────────┬────────────────────┐
│ - Menu1        │                    │
│ - Menu2        │   Content Area     │
│ - Menu3        │                    │
└────────────────┴────────────────────┘
```

## Cards

### Basic Card
```
┌─────────────────────────────────────┐
│ Title                               │

│                                     │
│ Content                             │
│                                     │
└─────────────────────────────────────┘

│ │        [Image Area]             │ │
│ └─────────────────────────────────┘ │

```

## Lists

### Basic List
```
┌─────────────────────────────────────┐
│ • List item 1                       │
├─────────────────────────────────────┤
│ • List item 2                       │
├─────────────────────────────────────┤
│ • List item 3                       │
└─────────────────────────────────────┘
```

### Action List
```
┌─────────────────────────────────────┐
│ Item 1                       [Edit] │
├─────────────────────────────────────┤
│ Item 2                       [Edit] │
└─────────────────────────────────────┘
```

## Modals/Dialogs

```
          ┌───────────────────────────┐
          │ Modal Title          [X]  │
          ├───────────────────────────┤
          │                           │
          │     Modal Content         │
          │                           │
          ├───────────────────────────┤
          │   [Cancel]  [Confirm]     │
          └───────────────────────────┘
```

## Status Indicators

### Loading
```

```

### Toast Messages
```
┌─────────────────────────────────────┐
│ ✓ Success message                   │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│ ✗ Error message                     │
└─────────────────────────────────────┘
```
